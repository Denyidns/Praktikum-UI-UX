Formulir ini berisi field kontak umum, validasi sederhana, dan input file. Cocok untuk kontak, pendaftaran, atau pengumpulan informasi pengguna.

Penggunaan
  1. Buka file `form.html` di browser.
  2. Untuk mengirim ke server, tambahkan atribut action="URL" method="POST" enctype="multipart/form-data" pada tag <form>.
  3. Sesuaikan field sesuai kebutuhan (tambahkan hidden input jika perlu).
